    - Expand/collapse span (shown only when children.length > 0): onClick toggles expandedIds (new Set copy), stopPropagation; shows "▼" when expanded, "▶" when collapsed; className "text-xs text-slate-400 w-4"

    - span with zone.name; className "text-sm" + (selectedId === zone.id ? " font-semibold text-slate-900" : "")
    - Row button onClick: setSelectedId(zone.id)
    - Recursively renders children when expandedIds.has(zone.id): {isExpanded && children.map(child => <ZoneTreeNode key={child.id} zone={child} depth={depth + 1} />)}
    - Note: ZoneTreeNode uses closure over world.zones, expandedIds, setExpandedIds, selectedId, setSelectedId — no prop drilling needed since it's a local function component

      - Field label="Zone Type": Pill with ZONE_TYPE_TONE tone
      - Field label="Requires Explicit Grant": {zone.requires_explicit_auth ? "Yes" : "No"}

      - Field label="Asset Owner": {unitName(zone.asset_owner_org_id as UnitId)}
      - If ancestors.length > 0: Field label="Path": spans of ancestor names joined by " › " text in text-slate-400

    - Right panel (col-span-2): Card title="Zone Detail" containing: if selected then <ZoneDetailPanel zone={selected} /> else <p className="text-sm text-slate-400">Select a zone to see details.</p>
