    Destructure { hubIndex, fedInbox, fedOutbox } from useWorld().
    const unitIds = Object.keys(UNITS) as UnitId[].
    const [selectedUnit, setSelectedUnit] = useState<UnitId>(unitIds[0]).
    // Note: selectedUnit is local state ONLY — does NOT dispatch to the store (D2-04 spirit / Pattern 6)

    Derive holdings: const holdings = hubIndex.filter(p => p.holdingUnit === selectedUnit).
    Derive inboxEntries: const inboxEntries = fedInbox[selectedUnit] ?? [].
    Derive outboxEntries: const outboxEntries = fedOutbox[selectedUnit] ?? [].

    Render structure — div className "space-y-4":
    h2 "Unit Console" — text-lg font-semibold.
    Card containing Field with label "Unit console" and a Select component:
      options = unitIds.map(id => ({ value: id, label: unitName(id) }))
      value = selectedUnit; onChange = setSelectedUnit.

    Holdings Card (title "Holdings (published pointers)"):
      If holdings.length === 0: p className "text-sm text-slate-400" with empty state text.
      Else: ul className "space-y-2" with li items for each pointer:
        li: span text-sm containing Pill tone="blue" unitName(p.holdingUnit); span className "text-slate-400" " holds "; Pill tone="amber" p.domain; span className "text-slate-400" " authz info".

    Inbox Card (title "Inbox (incoming detail requests)"):
      If inboxEntries.length === 0: p with empty state text.
      Else: div className "space-y-4" with one div per entry (className "space-y-2 border-b border-slate-100 pb-3 last:border-0 last:pb-0"):

          span className "text-slate-400": "→ " + entry.subjectId.
          Credential trust status: if entry.verifyResult.valid, Pill tone="green" "verified"; else Pill tone="red" "unverified" + MockTag component.
        Row 2: if entry.detailResult.decision is not null, render DecisionTrace component with result={entry.detailResult.decision}.

This makes the verify-before-trust path visible in the inbox (D2-10 / UI-SPEC release withheld reason).
        Use entry.seq as key.
