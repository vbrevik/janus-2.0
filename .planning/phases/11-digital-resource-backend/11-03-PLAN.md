<objective>
Expose the persisted domain through AuthGuard-protected endpoints: one aggregate `GET /api/digital-resources/world` (RSRC-BE-03), two issue POSTs (`/grants`, `/delegates`) that re-validate authority server-side via the ported `can_issue_resource_grant` (RSRC-BE-04), mount the domain, seed the 6-unit `seed.ts` fixtures into Postgres idempotently, and stop `seedWorld()` from hardcoding the digital-resource arrays (RSRC-BE-05).

Purpose: This is where the backend becomes the source of truth — the DB serves the dataset the frontend used to hardcode, and the issue path enforces the trust boundary server-side so a client cannot assert its own authority.
Output: real `handlers.rs`, the mount line, the seed migration, integration tests, and a de-hardcoded `seedWorld()`.
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → issue POST | Untrusted client claims authority to issue a grant/delegate |
| client → GET /world | Unauthenticated read of the full authorization dataset |
| seed migration → DB | Re-applied seed must not duplicate rows |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-08 | Elevation of Privilege | issue POST trusts client authority | mitigate | Server re-validates via can_issue_resource_grant from DB state before any INSERT; 403 on false (3 cases test-asserted) |
| T-11-09 | Elevation of Privilege | Expired/out-of-window delegate reused | mitigate | is_window_active inside can_issue_resource_grant rejects expired/out-of-window delegates |
| T-11-10 | Information Disclosure | Unauthenticated read of /world | mitigate | AuthGuard on get_world; unauthenticated GET returns 401 (test-asserted) |
| T-11-11 | Tampering | Duplicate issue creates duplicate rows | mitigate | uq_grant UNIQUE + ON CONFLICT DO NOTHING; duplicate issue yields no new row (test-asserted) |
| T-11-12 | Denial of Service | N+1 queries assembling /world | accept | Demo backend; 8 fixed parallel queries, small dataset (RESEARCH dismissed concurrency edge) |
| T-11-SC | Tampering | npm/pip/cargo installs | accept | No new packages (RESEARCH §Package Legitimacy Audit: N/A) |
</threat_model>
