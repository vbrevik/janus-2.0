| T-01-SC | Tampering | npm installs | accept | No new packages installed in this plan (entry + config only); package legitimacy gate not triggered. |

No unmitigated HIGH-severity threat. This is a frontend-only in-memory demo island; threats are proportional (mock-vs-real confusion, build leakage), not production-auth threats.
